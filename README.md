Deployment of a centralized infrastructure based on Microsoft Active Directory and TrueNAS.

## Technologies

- Windows Server
- Active Directory
- DNS
- GPO
- TrueNAS
- SMB
- iSCSI
- Hyper-V

## Features

- Centralized user management
- SMB network shares
- iSCSI storage
- Roaming profiles
- Automated drive mapping through GPO
